## Run Instruction
```
git clone https://github.com/meandevstar/jekyll-book-reading-ruby.git
cd jekyll-book-reading-ruby
gem install jekyll
```

Check if jekyll gem is installed
```
jekyll -v
```

Run static server
```
jekyll serve
```
